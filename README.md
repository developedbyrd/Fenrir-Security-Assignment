
Senrir Security — UI Assignment
================================

Overview
--------

React + Vite demo app showing scan results and vulnerability details. It's a small frontend exercise focused on UI components, layout, and local mock data.

Tech
----

- Vite + React + TypeScript
- pnpm used for package management

Quick start
-----------

- Install dependencies: `pnpm install`
- Run dev server: `pnpm dev`
- Build for production: `pnpm build`
- Preview build: `pnpm preview`

Project layout (high level)
---------------------------

- `src/pages/dashboard` — main views and pages
- `src/components` — UI components and small primitives
- `src/lib/mock-data.ts` — sample data used by the UI
